1. Etiquetas de texto

En esta práctica de Desarrollo Web utilizamos etiquetas HTML para dar formato al texto sin aplicar estilos CSS.

Ejemplo de etiquetas comunes: Negrita, Cursiva, Subrayado, Texto resaltado, Texto en tamaño reducido, Texto eliminado y Texto insertado.

También usamos abreviaturas como HTML y CSS, además de citas como: El lenguaje de la web es HTML.
