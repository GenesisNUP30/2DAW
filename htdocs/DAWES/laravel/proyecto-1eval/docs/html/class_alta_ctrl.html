App::Http::Controllers::AltaCtrl::alta ( )

Método principal para gestionar la creación de tareas.

Si se recibe un POST, filtra y valida los datos. En caso de errores, devuelve la vista 'alta' con los datos introducidos y mensajes de error. Si no hay errores, guarda la tarea en la base de datos y redirige a la página principal.

Si no se recibe un POST, carga la vista 'alta' con campos vacíos por defecto.

Returns
\Illuminate\View\View|\Illuminate\Http\RedirectResponse Retorna la vista de alta o redirige al listado.

◆ filtraDatos()

App::Http::Controllers::AltaCtrl::filtraDatos ( )
private

Filtra y valida los datos del formulario de alta de tareas.

Verifica que los campos obligatorios no estén vacíos, valida NIF/CIF, correo electrónico y teléfono, y asegura que la fecha de realización sea posterior a la actual.

Los errores se almacenan en Funciones::$errores con el nombre del campo como clave y el mensaje de error como valor.

Returns
void
