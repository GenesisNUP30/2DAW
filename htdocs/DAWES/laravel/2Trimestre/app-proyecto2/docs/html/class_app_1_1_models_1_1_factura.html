Detailed Description

Modelo que representa el documento legal de facturación emitido.

  • Este modelo actúa como un "snapshot" o captura de datos. Almacena de forma redundante los datos del cliente (nombre, CIF) y de la cuota (importe, concepto) para garantizar la integridad histórica: si un cliente cambia su CIF hoy, las facturas emitidas ayer deben conservar el CIF antiguo.
