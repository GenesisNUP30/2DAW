Detailed Description

Modelo que representa las obligaciones de pago de los clientes.

  • Gestiona tanto las cuotas mensuales recurrentes como las excepcionales. Incorpora el uso de Soft Deletes para permitir una "papelera de reciclaje" y mantener la trazabilidad contable.
