Detailed Description

Funciones utilitarias y validaciones para el sistema de gestión de tareas.

Esta clase contiene métodos estáticos que realizan:

  • Validaciones de NIF/CIF y teléfonos.
  • Validaciones de contraseñas y código postal.
  • Conversión y formato de fechas.
  • Generación de listas de provincias españolas.
  • Gestión de errores de validación.

Validar un NIF o CIF.

Comprueba si el valor introducido es un NIF o CIF válido. Para NIF verifica la letra final según el algoritmo español. Para CIF calcula el dígito/letra de control.

Parameters

Returns
