Detailed Description

Singleton para la conexión a la base de datos MySQL usando mysqli.

Esta clase implementa un patrón singleton para asegurar que solo haya una conexión activa a la base de datos durante la ejecución del proyecto. Proporciona métodos para:

  • Ejecutar consultas SQL.
  • Escapar valores para evitar inyección SQL.
  • Leer registros de resultados de consultas.
  • Obtener el último ID insertado.
