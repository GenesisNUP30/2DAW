Detailed Description

Gestión de la configuración avanzada de la aplicación.

Esta clase permite obtener y modificar configuraciones avanzadas almacenadas en la base de datos, como:

  • Límites de intentos de login
  • Duración de sesiones
  • Políticas de contraseñas
  • Items por página y tema de la aplicación

Implementa el patrón Singleton para asegurar que solo exista una instancia de configuración en toda la aplicación.
