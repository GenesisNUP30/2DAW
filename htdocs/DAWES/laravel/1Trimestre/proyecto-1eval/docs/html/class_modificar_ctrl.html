Detailed Description

Controlador encargado de la modificación de tareas existentes.

Este controlador gestiona el proceso completo de edición de una tarea:

  • Visualización del formulario de modificación.
  • Validación de los datos introducidos por el usuario.
  • Actualización de la tarea en la base de datos.

Solo los usuarios autenticados con rol de administrador pueden acceder a estas funcionalidades.

◆ actualizar()

App::Http::Controllers::ModificarCtrl::actualizar ( $id)

Actualiza los datos de una tarea existente.

Este método:

  • Comprueba que la tarea exista.
  • Filtra y valida los datos enviados mediante POST.
  • Si existen errores de validación, vuelve a mostrar el formulario con los datos introducidos y los mensajes de error.
  • Si los datos son correctos, actualiza la tarea y redirige a la página principal.

Exceptions

Symfony\Component\HttpKernel\Exception\NotFoundHttpException Se lanza si la tarea no existe.

◆ mostrarFormulario()

App::Http::Controllers::ModificarCtrl::mostrarFormulario ( $id)

Muestra el formulario de modificación de una tarea.

Este método:

  • Verifica que el usuario esté autenticado.
  • Verifica que el usuario tenga permisos de administrador.
  • Obtiene los datos de la tarea a partir de su identificador.
  • Envía los datos a la vista modificar.

Si la tarea no existe, se lanza una excepción HTTP 404.

int$idIdentificador único de la tarea a modificar.
Returns
\Illuminate\View\View Devuelve la vista modificar con los datos de la tarea.
Exceptions

Symfony\Component\HttpKernel\Exception\NotFoundHttpException Se lanza si la tarea no existe.
